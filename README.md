# qtpg

Example Qt and PostgreSQL connection

## This is command to create database **rtpq**


```sql
CREATE DATABASE rtpq
  WITH OWNER = postgres
       ENCODING = 'UTF8'
       TABLESPACE = pg_default
       LC_COLLATE = 'Turkish_Turkey.1254'
       LC_CTYPE = 'Turkish_Turkey.1254'
       CONNECTION LIMIT = -1;
```
